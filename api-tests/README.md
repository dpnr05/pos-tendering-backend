\# API Test Payloads



This folder contains sample request payloads for testing the POS Tendering Backend APIs.



\## Files

\- tender-single-request.json — Single tender payment

\- tender-split-request.json — Split tender scenario

\- tender-invalid-request.json — Validation / error scenario



These payloads can be used with:

\- Postman

\- Bruno

\- curl
